Weak duality explorer: max 2x + 3y

Multiply the constraints by a, b, c ≥ 0 and add them: the orange line t·x + w·y = d. Whenever t ≥ 2 and w ≥ 3, every feasible point satisfies 2x + 3y ≤ t·x + w·y ≤ d, so d bounds the optimum. Try to push d down to 23 — that is the dual problem.
