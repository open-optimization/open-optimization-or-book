\section{Introduction}

This document tests the basic alt text functionality using the modern LaTeX
tagging system. The key feature is the \texttt{alt=\{\}} option in
\verb|\includegraphics|.

\section{Image with Alt Text}

Here is an image with embedded alt text:

\begin{figure}[ht]
  \centering
  \includegraphics[width=0.6\linewidth, alt={A two-dimensional coordinate system showing a shaded polygonal region representing the feasible region of a linear program. The polygon is bounded by several linear constraint lines forming a convex shape.}]{LP-feasible-region.png}
  \caption{Linear Programming Feasible Region}
  \label{fig:lp-feasible}
\end{figure}

The alt text above will be embedded in the PDF structure and read by screen readers.

\section{Verifying Accessibility}

To verify alt text is embedded:
\begin{enumerate}
  \item Open the PDF in Adobe Acrobat
  \item View $\rightarrow$ Navigation Panels $\rightarrow$ Tags
  \item Find Figure tags and check their properties
\end{enumerate}
