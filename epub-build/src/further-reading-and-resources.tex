\chapter{Further Reading and Resources}

The following free textbooks and notes serve as useful alternative or supplementary references for the topics covered in this book.

\paragraph{\textbf{Optimization and Linear Programming}}
\begin{itemize}
\item \href{http://www.optimization-online.org/DB_FILE/2013/12/4161.pdf}{A First Course in Optimization} by Jon Lee
\item \href{https://web.stanford.edu/~boyd/cvxbook/}{Convex Optimization} by Boyd and Vandenberghe
\item \href{https://www.springer.com/gp/book/9783540306979}{Understanding and Using Linear Programming} by Matousek and G\"artner (free via Springer with university account)
\item \href{http://math.mit.edu/~goemans/18310S15/lpnotes310.pdf}{Linear Programming Notes} by Michel Goemans (MIT)
\end{itemize}

\paragraph{\textbf{Discrete Mathematics and Combinatorics}}
\begin{itemize}
\item \href{http://discrete.openmathbooks.org/dmoi3.html}{Discrete Mathematics: An Open Introduction} by Oscar Levin (CC-BY-SA)
\item \href{http://www.rellek.net/book/frontmatter-1.html}{Applied Combinatorics} by Keller and Trotter (CC-BY-SA 4.0)
\end{itemize}

\paragraph{\textbf{Mathematical Background}}
\begin{itemize}
\item \href{https://open.umn.edu/opentextbooks/textbooks/book-of-proof}{Book of Proof} by Richard Hammack (proof techniques)
\item \href{http://linear.ups.edu/html/fcla.html}{A First Course in Linear Algebra} by Robert Beezer
\end{itemize}

\paragraph{\textbf{Programming}}
\begin{itemize}
\item \href{https://open.umn.edu/opentextbooks/textbooks/a-byte-of-python}{A Byte of Python} (CC-BY-SA, free online textbook)
\item \href{https://coin-or.github.io/pulp/}{PuLP Documentation and Tutorials}
\end{itemize}

\noindent For optimization software, solver tutorials, and modeling language resources, see Appendix~\ref{app:software}. Additional example code is available at \url{https://github.com/open-optimization/open-optimization-or-examples}.
